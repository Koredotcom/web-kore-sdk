Kore Web SDK — kwsdk- utilities & typography

Layout utilities live in src/components/chatwindow/sass/utilities.scss. Typography scale (font size / line height for copy) lives in src/components/chatwindow/sass/typography.scss. Both are imported from chatWindow.scss. Prefixed classes use $kr-prefix (default kwsdk-) so they stay separate from CSS on the host page.

Note: This file is the full reference (open it in a browser). The kwsdk-* styles apply when the chat window bundle loads (for example after a webpack build that includes utilities.scss and typography.scss).

Why kwsdk-?

Class names use the kwsdk- prefix so they do not clash with global utility classes in the host application. Layout helpers live in utilities.scss; type scale in typography.scss — both ship with the chat window stylesheet via chatWindow.scss.

Quick start

Add multiple classes on any element inside the chat UI markup (same idea as Bootstrap, with the prefix):

<div class="kwsdk-d-flex kwsdk-justify-content-between kwsdk-align-items-center kwsdk-gap-2 kwsdk-mb-3">
  …
</div>

Spacing scale: sizes 0–5 map to tokens in $kr-spacers inside utilities.scss. auto is allowed for margin utilities only, not padding.

Typography (typography.scss)

Semantic text sizes: one class per step, color from branding.

Color: all steps use var(--sdk-gloabl-text-color) (spelling matches src/templatemanager/base/variables.scss on :root). Ensure that stylesheet (or equivalent tokens) is present so the variable resolves.

Pattern: kwsdk-text-<size> — prefix comes from $kr-prefix in typography.scss (same default as utilities).

<p class="kwsdk-text-md">Body copy</p>
<p class="kwsdk-text-sm semibold">Caption, semibold</p>

Weight helpers medium, semibold, and bold are unprefixed modifier classes on the same element as a kwsdk-text-* class (they are nested in SCSS under each typography rule).

Optional modifiers (combine with any kwsdk-text-*): medium (500), semibold (600), bold (700). Default weight uses var(--font-weight-normal-Regular, 400) from the headingTextDisplay mixin.

Override the scale before importing typography.scss by redefining $typography-variants (map with a text key; each size entry supports size, line-height, and optional spacing for letter-spacing).

Margin and padding

Pattern: kwsdk-<prop>-<0–5|auto>. Uses !important.

  • Margin: kwsdk-m-*, kwsdk-mt-*, kwsdk-mb-*, kwsdk-ms-*, kwsdk-me-*, kwsdk-mx-*, kwsdk-my-* (ms/me = inline start/end).
  • Padding: kwsdk-p-*, kwsdk-pt-*, kwsdk-pb-*, kwsdk-ps-*, kwsdk-pe-*, kwsdk-px-*, kwsdk-py-* — no auto token for padding.

Text & font utilities (utilities.scss)

These are layout-oriented helpers (alignment, transform, weight), not the typography scale above. Some names start with kwsdk-text- but they are separate from kwsdk-text-md / kwsdk-text-sm typography classes.

Class Effect
kwsdk-text-start, kwsdk-text-end, kwsdk-text-centerText alignment
kwsdk-text-lowercase, kwsdk-text-uppercase, kwsdk-text-capitalizetext-transform
kwsdk-fw-bold, kwsdk-fw-semibold, kwsdk-fw-normal, kwsdk-fw-lightFont weight
kwsdk-fst-italic, kwsdk-fst-normalFont style
kwsdk-text-wrap, kwsdk-text-nowrap, kwsdk-text-truncateWhitespace / ellipsis

Overflow

kwsdk-overflow-{auto|hidden|visible|scroll} and the same suffixes on kwsdk-overflow-x-*, kwsdk-overflow-y-*.

Position and inset

  • kwsdk-position-static, kwsdk-position-relative, kwsdk-position-absolute, kwsdk-position-fixed, kwsdk-position-sticky
  • kwsdk-top-*, kwsdk-bottom-*, kwsdk-start-*, kwsdk-end-* with 0, 50, 100, auto

Width and height

kwsdk-w-25, 50, 75, 100, auto and the same keys for kwsdk-h-*. Also kwsdk-mw-100, kwsdk-mh-100.

Border and radius

kwsdk-border, kwsdk-border-0, kwsdk-border-top, kwsdk-border-bottom; kwsdk-rounded, kwsdk-rounded-0 … kwsdk-rounded-3, kwsdk-rounded-circle, kwsdk-rounded-pill.

Shadow

kwsdk-shadow-none, kwsdk-shadow-sm, kwsdk-shadow, kwsdk-shadow-lg.

Visibility and z-index

kwsdk-visible, kwsdk-invisible. Z-index: kwsdk-z-0, 1, 2, 3, 4, 5, 10, 50, 100, 1000, 1020, 1030, 1040, 1050.

Interaction

kwsdk-user-select-all, kwsdk-user-select-auto, kwsdk-user-select-none.

kwsdk-pe-none and kwsdk-pe-auto control pointer-events, not padding. Padding end uses kwsdk-pe-0 … kwsdk-pe-5 (padding).

Customizing in SCSS

Before importing the partials (or in a file imported earlier), you can override:

// Shared prefix (set once before both imports if you want them in sync)
$kr-prefix: 'kwsdk-';

// utilities.scss — from another file in the same folder, for example:
// Default $kr-spacer is 0.25rem (one 4px step). Larger $kr-spacer scales all tiers.
// $kr-spacer: 0.5rem; // optional: 8px, 16px, 24px, …
// $kr-spacers map can be redefined if needed
@import 'utilities.scss';

// typography.scss
// $typography-variants: ( 'text': ( ... ) );
@import 'typography.scss';

!default is used on $kr-prefix, $kr-spacer, $kr-spacers (utilities), and $typography-variants (typography).
